# Proyecto final - React

Este repositorio contiene la versión final del proyecto de React para el curso de Talento Tech. Es una aplicación web que permite a los usuarios explorar una lista de productos, ver detalles de cada producto, iniciar sesión (autenticación), y gestionar un carrito de compras. La aplicación está construida con React, React Router y context API para el manejo de estados.

## Cómo ejecutar
1. Clonar el repositorio
```powershell
git clone <repo-url>
cd proyecto-final
```
2. Instalar dependencias
```powershell
npm install
```
3. Ejecutar en modo desarrollo
```powershell
npm run dev
```
4. Abrir en el navegador para ver la aplicación
